“整理一下c++中的输入”

发表于 2019-01-25 | 分类于 c++ |

主要是整理一下常用的输入输出方法 （不仅仅限于cin和scanf，例如getline等的详细用法）

1、 cin

最普通的cin函数是直接从缓冲区取数据。int a ； cin>>a 遇到结束符就结束，例如空格 回车就不读入 类似的还有数组，当键盘输入回车空格时，a[ ]不存储空格或回车。 当输入一个字符串时，cin遇到空格和回车就停止读入。

2、 cin.get

（1）char a 可以用 a=cin.get();或cin.get(a);

（2）cin.get 可以用于接受字符串，可以接收空格
cin.get(字符数組名,接收字符数目) char a[20];cin.get(a,20);//接收19个字符，并加上一个’\0’。

3、 cin.getline()

与cin.get类似 cin.getline有3个参数，分别是字符数组的名字（不能用string），读取的长度，结束字符，一般结束字符为’\0’时可省略，不读取结束字符。唯一区别是cin.get遇到结束符后指针不变，而cin.getline的指针移到结束符下一位。char m[20]; cin.getline(m,5); // cin.get(m,5);

4、 getline()

可以接收空格并输出，其头文件与以上不同，包含在#include< string > 中，用法也是针对string变量，可以读取空格，遇到’\n’结束 ，例如 string m； getline（cin,m);

5、 gets()

头文件为#include<stdio.h>，对象为字符数组 。例如char a[20]; gets(a); //此处不能写出a=gets();
